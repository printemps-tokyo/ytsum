# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1 as its code of conduct.

Everyone participating is responsible for keeping this an open, harassment-free
environment. Discriminatory or insulting behavior and harassment are not
tolerated.

Report violations to info@printemps.tokyo. Reports are handled confidentially.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
